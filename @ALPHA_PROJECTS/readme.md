# 🎮 Hangman Game (Tech Edition)

A console-based Hangman game built using Python.  
This project demonstrates core programming concepts like loops, conditional statements, and list manipulation.

---

## 🚀 Features
- Random tech-themed words
- Limited number of guesses
- Input validation (only one letter allowed)
- Dynamic word display using blanks
- Win/Lose conditions with messages 🎉💀

---

## 🛠️ Technologies Used
- Python 3
